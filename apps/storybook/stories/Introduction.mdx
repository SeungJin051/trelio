# Trelio 스토리북

이 스토리북은 Trelio 프로젝트의 UI 컴포넌트 라이브러리를 문서화한 공간입니다.

## 스토리북 구성

이 스토리북은 다음과 같은 카테고리로 구성되어 있습니다:

- **기본 요소**: 버튼, 입력 필드, 타이포그래피 등의 기본 UI 요소
- **레이아웃 컴포넌트**: 헤더, 푸터, 사이드바 등의 레이아웃 관련 컴포넌트
- **UI 컴포넌트**: 카드, 모달, 알림 등 자주 사용되는 복합 컴포넌트

## 스토리북 사용 방법

각 컴포넌트는 다음과 같은 정보를 포함하고 있습니다:

1. **기본 사용법**: 컴포넌트의 기본적인 사용 예시
2. **Props 테이블**: 컴포넌트에서 사용 가능한 모든 속성과 그 설명
3. **변형**: 다양한 상태, 크기, 스타일의 변형 예시
4. **상호작용**: 클릭, 호버 등의 상호작용 테스트

스토리북의 좌측 메뉴에서 원하는 컴포넌트를 선택하여 탐색할 수 있습니다.
